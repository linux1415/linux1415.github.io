Standard Encrypter Instructions



  1. Choose a key file from your hard drive or an external device such CD/DVD/USB. After this, it is recommended that you wait a few seconds before doing anything else while the key loads.
  2. Choose your security setting of "Normal" or "Kill Switch at Repetition". "Kill Switch at Repetition" is the default setting and you should leave it as such unless the program prompts you to use the "Normal" setting; you will only be prompted to do such if the message is very large.
  3. Type in your message or paste it in from elsewhere.
  4. Click "Encrypt Level 1"; any subsequent time button is clicked a new ciphertext will be generated.
  5. If Level 2 Encrytion is desired (this encryption is recommended if the same key is being used often; in addition it will enable secure communication even if digital key is compromised. For added security, do not store number and passcodes on computer; print them out.):
    • Type in a number code. Each individual number can be a different size if desired. I would not go over 4 digits, though.
    • Type in a passcode. Passcode can be any length but can only be composed of lowercase letters, capital letters, and numbers. A length of 12 characters will be most secure; a length of more than 12 characters will work but won't make ciphertext more secure.
    • Click "Encrypt Level 2"
    • The encrypter and decrypter must have the exact same number code and passcode.
  6. Ciphertext may be saved to a textfile using the "Save Ciphertext To Textfie" button if using the Google Chrome browser. Otherwise, ciphertext may be simply copied and pasted to wherever is desired.
