Information



  • Choose Key: key must be chosen from hard drive of computer; this will take several seconds to load into your browser.
  • Choose Key Size: this is the number of possibilities per individual plaintext character.
  • Choose Security Setting: "Kill Switch at Collision" means there will be no repeated individual ciphertexts (always use unless message has about 75% as many words as the key size; if the key size is 1000, then use if message is up to 750 words); for longer messages use the "Normal" setting. If unsure of what to do, always use "Kill Switch at Collision".
  • Save to Textfile: this feature only works if using Google Chrome webbrowser.
  • How Secure is this System?: If a different key is used for each message, and "Kill Switch at Collision" feature is on, all messages should be unbreakable. The more a key is used, the less secure it will be. The larger the key size, the longer it can be used securely. Key sizes up to 10000 have been tested successfully.
  • Is the fact that this page is on the internet going to inhibit the security of this system?: No, all processing is client side. In other words, your browser is doing the encryption, not the web server that this page is on. You are simply downloading the html code from the web server; everything else that happens has nothing to do with the internet.
